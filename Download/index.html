Helix2D

交织创造潜力

下载代码

你可以直接在此下载Helix2D的静态链接库与头文件，或通过访问GitHub下载所有源代码

最新版本：Helix2D-v0.4.1
Git：https://github.com/FriesPeter/Helix2D.git

历史版本

  • Helix2D-v0.4.1

    • New
      • 新增Random类，用于随机数运算，可以范围内取随机数，也可以以概率返回bool值
      • Window类新增Tick方法
    • Fix
      • 修复addPainter方法无法正常使用的问题
      • 修复Painter类isOvelap方法无法正常检测缩放后的Painter的问题

  • Helix2D-v0.4.0

    • New
      • Window类现已支持透明窗口
      • Painter类新增Name属性，让开发者能更好的通过Name查找对象
      • 新增Logger类，Logger会在一些情况下打印消息提示开发者，不需要打印也可以选择分别关闭各种消息种类：Error，Warning，Message
    • Refactor
      • 重构部分底层代码

  • Helix2D-v0.3.2

    • Fix
      • 修复Renderer类析构时无法正常释放设备相关资源的问题
      • 修复Painter类无法正常添加Module的问题

  • Helix2D-v0.3.1

    • Fix
      • 修复Window类setFPS无法工作的问题
      • 修复Window类析构函数无法正常移除Painter的问题

  • Helix2D-v0.3.0

    • New
      • 新增Mask类，Painter类的子类，全称Mask遮罩，用于遮蔽画面，它能将子画家在划定区域内的部分隐去，显现出原有的画面
    • Refactor
      • 重构Window类底层窗口循环代码，大大优化程序性能

  • Helix2D-v0.2.2

    • Fix
      • 修复使用Sprite类与Image类在程序结束时出现内存访问冲突的问题

  • Helix2D-v0.2.1

    • Fix
      • 修复Initialize类致命错误，解决了Initialize类无法被包含进lib的问题

  • Helix2D-v0.2.0（不可用）

    • New
      • 新增Math类，用于数学计算（弧度转角度等），但大部分计算方法仍用C库数学库（cmath），如：cosf，sinf，powf，sprt等等，Math类只提供cmath部分没有的简单计算
    • Fix
      • 修复removePainter与removeModule可能发生的内存错误
      • 修复Input无法检测L/R Ctrl/Alt/Shift按键的问题

  • Helix2D-v0.1.1（不可用）

    • Fix
      • 修复moveAhead，addVelocityAhead方向错误的问题

  • Helix2D-v0.1.0（不可用）

    • Refactor

      • 重构Image类底层实现

    • New

      • Painter类新增Velocity，Mass等物理量及控制方法
      • Painter类新增moveAhead，addVelocityAhead方法，用于以特定方向移动或加速
      • Painter类新增removePainter，removeModule方法，用于移除画家或模块
      • 新增大部分类定义注释
      • 等等新功能

    • Changed

      • getTopLeftPos系列函数改名为getUpperleftPos

  • Helix2D-v0.0.1

    • New
      • 第一个内测版本
