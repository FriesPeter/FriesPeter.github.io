Window::Window

  • 语法
  • 参数
  • 返回值
  • 备注

该方法用于构造一个Window类对象

语法

参数

  • [in] title
    用于指定窗口的标题
    若不指定参数，则默认为”Helix2D”
  • [in] width
    用于指定窗口的宽度
    若不指定参数，则默认为600
  • [in] height
    用于指定窗口的高度
    若不指定参数，则默认为600.
  • [in] parent
    用于指定窗口的父窗口
    当该值为nullptr时，则视为没有父窗口
    若不指定参数，则默认为nullptr
  • [in] fps
    用于指定期望的窗口帧率
    Window类会以1/帧率每秒的时间为间隔更新窗口
    若不指定参数，则默认为60

返回值

无

备注

该方法会初始化整个Window对象，并将所有传入的信息保存到对象中，同时动态创建渲染器（Renderer）。随后将地址信息存入[private] allWindow容器中进行管理，随后创建窗口线程，开始窗口循环

注意：构造对象不代表窗口创建，实际上，窗口创建在窗口线程中执行
所以实际的窗口创建在构造对象之后才开始
